# CATLASS Matmul+Gelu 算子设计文档

## 1. 需求分析

### 1.1 功能说明

实现矩阵乘法后接 GELU 激活函数的融合算子，用于深度学习模型中的高效计算。算子将矩阵乘法和激活函数融合在一个 Kernel 中执行，减少内存访问开销，提高计算效率。

### 1.2 数学公式

算子执行以下计算：

$$out = Gelu(A \times B)$$

其中，GELU 函数的计算公式为：

$$Gelu(x) = 0.5 \times x \times (1 + Tanh(\sqrt{2/\pi} \times (x + 0.044715 \times x^3)))$$

或其近似形式：

$$Gelu(x) \approx \frac{x}{1 + e^{-1.595769 \times (x + 0.044715 \times x^3)}}$$

### 1.3 变量说明表

| 变量 | 类型 | 含义 | 约束条件 |
|------|------|------|----------|
| A | half (FP16) | 输入矩阵 A | 形状：M × K，非转置 |
| B | half (FP16) | 输入矩阵 B | 形状：K × N，非转置 |
| out | half (FP16) | 输出矩阵 | 形状：M × N |
| M | int64_t | 矩阵 A 的行数，矩阵 out 的行数 | M > 0 |
| K | int64_t | 矩阵 A 的列数，矩阵 B 的行数 | K > 0 |
| N | int64_t | 矩阵 B 的列数，矩阵 out 的列数 | N > 0 |

## 2. 注意事项

1. 该实现基于 CATLASS 框架，需要确保 CATLASS 仓库正确配置
2. 算子仅支持 FP16 数据类型的输入和输出
3. 矩阵 A 和矩阵 B 均为非转置布局
4. 实现中使用了 float 作为中间计算结果类型，以提高计算精度
5. 根据 M 和 N 的大小自动选择不同的任务调度策略，优化性能
6. 支持 Ascend 910B 硬件平台
